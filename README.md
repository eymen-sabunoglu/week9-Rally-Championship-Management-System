# Week 9 Rally Championship Management System

This project is a Java-based Rally Championship Management System.

## Files
- Driver.java
- RallyCar.java
- GravelCar.java
- AsphaltCar.java
- RaceResult.java
- RallyRaceResult.java
- ChampionshipManager.java
- ChampionshipStatistics.java
- Main.java

## How to run
Compile:
javac *.java

Run:
java Main
